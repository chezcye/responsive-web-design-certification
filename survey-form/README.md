# Survey Form

This project is part of the freeCodeCamp Responsive Web Design Certification.

## Objective
Build a responsive survey form using semantic HTML.

## Features
- Form title and description
- Required name and email fields
- Number input with min and max validation
- Dropdown selection
- Radio buttons
- Checkboxes
- Textarea for additional feedback
- Submit button

## Technologies Used
- HTML5

## What I Learned
- How to structure accessible form elements
- How to connect labels to inputs correctly
- How to build and style a responsive form layout

## Future Improvements
- Improve visual styling
- Add stronger spacing and hierarchy
- Refine mobile responsiveness
